# Number Guessing Game 🎯

A simple Java console-based Number Guessing Game where the computer generates a random number between 1 and 100, and the user tries to guess it within limited attempts.

## 📌 Features

- Generates a random number between 1 and 100
- User gets 5 attempts to guess the correct number
- Gives hints:
  - Number is greater than your guess
  - Number is less than your guess
- Displays success message when the correct number is guessed
- Displays the correct number if all attempts are exhausted

## 🛠️ Technologies Used

- Java
- Scanner Class
- Math.random()
